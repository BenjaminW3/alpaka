Detailed Description

template<typename TDim>
class alpaka::accs::omp::omp4::cpu::detail::AccCpuOmp4< TDim >

The CPU OpenMP4 accelerator.

This accelerator allows parallel kernel execution on a CPU device. It uses CPU OpenMP4 to implement the parallelism.
