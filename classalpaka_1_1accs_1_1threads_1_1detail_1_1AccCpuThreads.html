Detailed Description

template<typename TDim>
class alpaka::accs::threads::detail::AccCpuThreads< TDim >

The CPU threads accelerator.

This accelerator allows parallel kernel execution on a cpu device. It uses C++11 std::threads to implement the parallelism.
