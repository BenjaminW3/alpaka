Detailed Description

template<typename TDim>
class alpaka::accs::serial::detail::AccCpuSerial< TDim >

The CPU serial accelerator.

This accelerator allows serial kernel execution on a cpu device. The block size is restricted to 1x1x1 so there is no parallelism at all.
